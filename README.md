## Getting Started

If you want your own copy just follow this steps on Linux

Clone the repo
```bash
git clone https://github.com/adkoprek/asm
```

Go into the project your are interested
```bash
cd <folder>
```

Build the project and enjoy
```bash
./build.sh
./bin/<project>
```

### Prerequisites

  - NASM compiler
  - GNU linker
